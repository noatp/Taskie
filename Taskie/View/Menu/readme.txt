the templates are for create a VC+ViewModel combo

How to use:

- copy 3 files TemplateInfo.txt, VC.txt and ViewModel.txt into directory below (create any directory if missing)
~/Library/Developer/Xcode/Templates/FileTemplates/Panto/VC+ViewModel.xctemplate

- rename TemplateInfo.txt to TemplateInfo.plist

- rename VC.txt to VC.swift

- rename ViewModel.txt to ViewModel.swift

- restart Xcode

- cmd+n, find Panto section.
